# Tenets

## 行動指針

### 1. 自分自身のスキル・経験に投資する
周りではなく、自分自身のスキルと経験の向上に時間とリソースを投じる。

### 2. 80/20の法則を活用
最も効果的な20%の活動に、時間とお金の80%を集中投資する。

### 3. 意思決定の3つのチェックポイント
#### クリティカル性の確認
- その方法は本当に必要不可欠か？
- 代替手段はないか？

#### コスト最適化
- より低コストでトライできる方法はないか？
- 最小限の投資で検証できないか？

#### 経験価値の重視
- 失敗しても経験として学べることをやる
- リスクを取って挑戦することで成長する
